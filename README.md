### 📝 Sobre:
- Projeto feito com Java script. Uma calculadora com as 4 principais operações matemátias (adição, subtração, multiplicação e divisão).
- Opção de tema escuro e claro.

### ⚙️ Tecnologias:
- JavaScript
- HTML
- CSS
